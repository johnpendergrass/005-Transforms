aBoUt

Feb 11 - Feb 17, 2015

This is my first attempt at a serious
demo web site.

It uses -html5- and -css3- only,
with no javascript or other code.

Therefore it really is a series
of choreographed mini 'movies',
with little-to-no interaction.

Basically just hover on things to 
see what happens.

The -html- and -css- code is 
fairly well documented.

I think it has turned out
fairly well.  As I learned more
about 2D transforms it makes me
want to learn about the 
@animation stuff, and 3D 
transforms.  Done.
Next project.
