SCALE

The dirt bike doesn't work so well.
I wanted it to seem to roar past, 
just missing the train.

The train demo uses both -scale-
and -translate- on all items
(train, puffs of smoke, dirt bike).
The train has to move down the 
tracks (using a y axis translate), 
the puffs of smoke must appear to 
rise, etc.

The puffs of smoke are intially 
hidden behind the train, and only 
appear when their y axis translate
clears the train.

The word -SCALE- above shows that
if you set the scale to -0- then
the oject effectively disappears.

css3 snippet

/* load train very small (.05 scale) */

#train {
   position: absolute;
   left: 100px;
   top: -212px;
   transform-origin: center center;
   transform: scale(.05,.05);
   transition: 4s ease-in-out .5s;
}

article:hover #train {  /* make big and move down */
   transform: translate(20px, 25px) scale(1.15, 0.8);
}
