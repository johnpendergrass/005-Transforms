css 2D Transforms

OK, I know you really want to 
click in those squares and move 
the puzzle pieces around.

But that's not gonna happen.

This site is strictly -html5- and
-css3-, with no javascript or other 
code.  So just clicking and hovering. 

Not even those funny little css3
'@' things, which appear to offer
an intriguing mix of animation and 
other possibilities.

This is also just the 2D tranforms.
There are bunch of 3D ones that 
are not used in this project.

So, no real interactivity here -
unless you count 'hovering'... 

I hope you enjoy these demos.

html5 snippet

<!-- load puzzle pieces -->

<div id="puzzle">
  <img id="puzzle-01" class="puzzle" src="puzzle-01.png">
  <img id="puzzle-02" class="puzzle" src="puzzle-02.png">

css3 snippet

/* puzzle piece 08 - initial position */

#puzzle-08  {
  top: 254px ;
  left: 381px ;
  transition: all .20s linear 4.25s; /* this piece starts
                                        after 4.25 secs */
}

/* then move it when hovering over #puzzle element */

#puzzle:hover #puzzle-08 {
  transform: translate(0px, -127px);
}
