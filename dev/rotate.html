ROTATE

Remember this is just a 2D rotate
(ie. in the plane), so you cannot
get too fancy with turning over
cards or anything.

This demo uses pretty much only
the -rotate- css command.  After the 
cards are all positioned on top of
one another they are fanned into
position, each at their own angle, 
each with their own delay.

The top card was tricky to get to
fade and then reappear after the 
fan was finished.

The 'ROTATE' word above tries to
demo a propeller effect.

The 'rotate' word in the title uses
both -rotate- and -translate- to
cause it to slowly move outwards as
it spins, making a nice spiral
effect.

css3 snippet

/* the -.cards- class puts all the cards one on top of 
   each other, with a implied angle of -0deg- to start */

.cards {
	position: absolute;
	left: 367px;
 	top: 80px;
	transform-origin: bottom center;
	transition: 2s ease-out 1s;
}

/* when hovering, fan out the cards */

article:hover #card01 { transform: rotate(24deg); }
