SKEW

Most of these effects work fine on 
modern browsers.  Some seem a bit
flakey, strangely enough mainly
on Firefox.  IE seems good.  This
project was developed on Chrome,
in Feb 2015, so that is the browser
era it is designed for.

The -skew- effect gets very bizarre
when pushed past it's normal use.
Getting above -45deg- on either
axis generates strange stuff.
See the title word -skew- for an
example.

This starship demo uses all the 
transforms - rotate, scale,
translate and skew.  Skew is used
modestly to help 'flatten' out the 
fleet as it goes faster - 
to warp perhaps?

/* (right facing have a similar .ship-right) */

   position: absolute;
   top: 700px;
   left: -350px;
   transform-origin: center center;
   transform: skew(0deg) rotate(0deg);
}
