    ROTATE

    (2D transform)


    The 'rotate' transform is used to apply 2D rotation to a screen element.


    Format: 'transform: rotate(90deg);'


    If applied directly to the element the transform takes place immediately.


    If applied to a 'transistion' (ie. :hover or such) to the element, it will be animated according to the transition properties.


    The cards on the right each have a unique 'rotate' and transition applied to them, resulting in the card fanning animation. See the 'html' and 'css' code for details.
